Series vs Parallel Circuits

Class 10
Bulbs (4 Ω each)
Series 2 Parallel 2

What's going on?

Both boards run from the same battery. On the series board the bulbs sit in one loop, like a string of Diwali lights — the same current squeezes through every bulb, and they share the battery's voltage. Add more bulbs and every bulb dims.

On the parallel board each bulb has its own branch getting the full battery voltage — add more and they all stay just as bright. The battery simply pushes out more total current. Your home is wired this way, so the fridge doesn't dim when you switch on the fan.

And a fused bulb? In series it breaks the only path — the whole string dies. In parallel, only its own branch stops.

Formula

Series: Req = R₁ + R₂ + … (same I)
Parallel: 1/Req = 1/R₁ + 1/R₂ + … (same V)
